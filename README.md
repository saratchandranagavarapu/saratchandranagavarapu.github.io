# Sarat Chandra Nagavarapu academic website

This version uses an AcademicPages/Minimal-Mistakes-inspired layout: top navigation, left author profile, no vertical separator between the profile panel and content, and lighter grey typography.

Upload these files to your hosting folder, usually `htdocs`:

- `index.html`
- `research.html`
- `pub.html`
- `presentations.html`
- `experience.html`
- `ach.html`
- `assets/style.css`
- `data/publications.csv`

Also place these image/CV files in the same folder as `index.html`:

- `Sarat.png`
- `Sarat_CV.pdf`
- `linkedin.png`
- `google_scholar.jpg`
- `ResearchGate_Logo.png`
- `dblp.png`
- `orcid.png`
- `SciProfiles.png`
- `Publons.jpg`

`pub.html` is the only publications page in this version. `publications.html` and `projects.html` are not included.
